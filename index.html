1. ¿Qué es un Sistema Operativo?

Es el software principal que gestiona los recursos del hardware y proporciona servicios a otros programas.

2. Componentes de un Sistema Operativo:

Núcleo (kernel) Gestor de procesos Gestor de memoria Sistema de archivos Interfaz de usuario Controladores de dispositivos

3. Tareas del Sistema Operativo:

Administración de recursos Control de procesos Gestión de archivos Gestión de entrada/salida Seguridad y control de acceso

4. Funciones de un Sistema Operativo:

Coordinar el uso del hardware Ejecutar y controlar programas Proveer una interfaz al usuario Manejo de errores

5. Características de un Sistema Operativo:

Multitarea Multiusuario Seguridad Estabilidad Interfaz gráfica o de línea de comandos

6. Tipos de Sistema Operativo:

Monousuario vs. Multiusuario Monotarea vs. Multitarea Centralizado vs. Distribuido En tiempo real

7. Sistemas Operativos para PC:

Microsoft windows MacOS Linux FreeBSD Chrome OS Solaris Haiku ReactOS

8. Sistemas Operativos Móviles:

Android iOS HarmonyOS

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones:

Se elige según capacidad de hardware (PC, servidor, móvil) y el tipo de software a ejecutar (ofimática, edición, servidores, juegos, etc.).

10. Compatibilidad de los Sistemas Operativos:

Depende del hardware, programas y controladores disponibles para ese sistema. Algunos sistemas operativos solo funcionan con ciertos dispositivos o aplicaciones.

11. ¿Cuál es la función básica de un Sistema Operativo?

Gestionar los recursos del sistema y facilitar la interacción entre el usuario y el hardware.

12. Mencione tres Sistemas Operativos más utilizados:

Windows Android macOS

13. Menciona algunas tareas que realizan los Sistemas Operativos:

Ejecutar aplicaciones Administrar memoria Controlar periféricos Garantizar la seguridad del sistema

14. ¿Qué es un sistema operativo multitareas?

Es capaz de ejecutar múltiples tareas o procesos al mismo tiempo.

15. ¿Qué es un sistema operativo multiusuario?

Permite que varios usuarios utilicen los recursos del sistema simultáneamente o en diferentes sesiones.

16. ¿Cómo funciona un sistema operativo?

Actúa como intermediario entre el hardware y el software, coordinando las operaciones y gestionando recursos.

17. ¿Cómo se utiliza un sistema operativo en un entorno gráfico?

Mediante interfaces visuales como ventanas, iconos y menús, controlados por mouse y teclado.

18. ¿Cuándo inicia el funcionamiento del Sistema Operativo?

Al encender el equipo, después del proceso de arranque (boot), el BIOS/UEFI carga el sistema operativo.

19. Mencione las dos interfaces que forman parte del Sistema Operativo:

Interfaz gráfica de usuario (GUI) Interfaz de línea de comandos (CLI)

20. ¿Qué es un proceso en entorno de los sistemas operativos?

Es una instancia en ejecución de un programa que el sistema operativo gestiona de forma individual.

21. ¿Qué es un sistema operativo distribuido y centralizado?

Distribuido: opera en varios equipos interconectados, gestionando recursos de forma coordinada. Centralizado: todos los recursos se gestionan desde un único sistema central.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos:

NTFS (Windows) ext4 (Linux) APFS (macOS) FAT32 / exFAT (compatibles entre sistemas)

23. Mencione otras de las características actuales de los sistemas operativos:

Soporte para virtualización Actualizaciones automáticas Alta seguridad y cifrado Integración en la nube

24. ¿Qué permite la exigencia de muchos controladores en un sistema operativo?

Permite la compatibilidad con una gran variedad de dispositivos de hardware (impresoras, tarjetas gráficas, cámaras, etc.).
